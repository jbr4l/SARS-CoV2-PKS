# SARS-CoV2-PKS

DATA SETS PROVIDED:

1) RKI_COVID19_0716.csv
The RKI dataset downloaded from https://npgeo-corona-npgeo-de.hub.arcgis.com/ on the 16th July 2020.
Documentation: https://npgeo-corona-npgeo-de.hub.arcgis.com/datasets/dd4580c810204019a7b8eb3e0b329dd6_0

2) Global_Mobility_Report_World.csv
Global mobility data from Google downloaded from https://www.google.com/covid19/mobility/ on the 14th July 2020.
Documentation: https://www.google.com/covid19/mobility/data_documentation.html?hl=en

3) Global_Mobility_Report_Germany.csv
Mobility data in Germany from Google downloaded from https://www.google.com/covid19/mobility/ on the 14th July 2020.
Documentation: https://www.google.com/covid19/mobility/data_documentation.html?hl=en
